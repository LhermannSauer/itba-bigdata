# Trabajo Final - Herramientas Para la Extracción y Análisis de Grandes Volumenes de Datos

## Introduction

This project builds a reproducible Big Data + MLOps pipeline on Databricks that processes the Amazon Reviews dataset, produces models (sentiment / recommender / ranking), tracks experiments with MLflow, and powers a Retrieval-Augmented Generation (RAG) chatbot that can answer product.

The repo is designed for reproducibility, CI/CD via GitHub Actions, and clean Databricks integration.
